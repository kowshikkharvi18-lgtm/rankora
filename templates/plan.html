💻 Computer Awareness — Top 10 Topics
  • Number system conversions (Binary↔Hex↔Octal↔Decimal) — 3-4 questions guaranteed every year
  • Logic gates truth tables + De Morgan's laws (NAND/NOR = universal gates)
  • RAM vs ROM vs Cache vs Virtual memory — speed hierarchy: Cache > RAM > ROM > HDD
  • OSI model 7 layers — mnemonic: "Please Do Not Throw Sausage Pizza Away"
  • TCP vs UDP, HTTP=port 80, HTTPS=port 443, FTP=21, SMTP=25, DNS, DHCP
  • DBMS: ACID properties, normalization 1NF/2NF/3NF, Primary key vs Foreign key
  • Data structures: Stack(LIFO), Queue(FIFO), time complexities — Merge/Heap/Quick = O(n log n)
  • OS: Deadlock (4 Coffman conditions), SJF scheduling, paging vs segmentation, thrashing
  • OOP: Encapsulation, Inheritance, Polymorphism, Abstraction — definitions
  • Full forms: CPU ALU RAM ROM URL HTML CSS API GUI CLI USB PDF BIOS SDLC IDE
📐 Quantitative Analysis — Top 10 Topics
  • Logarithm rules: log(ab)=log a+log b, log(a/b)=log a-log b, log(aⁿ)=n·log a, log(1)=0
  • Trigonometry: sin/cos/tan values for 0°,30°,45°,60°,90° — memorize the table once
  • Permutation nPr = n!/(n-r)! and Combination nCr = n!/(r!(n-r)!) — "arrange"=P, "choose"=C
  • Probability: P(A)=favorable/total, P(A)+P(A')=1, P(impossible)=0, P(certain)=1
  • Sets: n(A∪B) = n(A)+n(B)-n(A∩B), number of subsets = 2ⁿ
  • AP: nth term = a+(n-1)d, Sum = n/2·(2a+(n-1)d)
  • Statistics: Mean=sum/n, Median=middle value, Mode=most frequent, Range=max-min
  • Quadratic formula: x = (-b ± √(b²-4ac)) / 2a, Discriminant D=b²-4ac
  • Geometry: Area circle=πr², triangle=(1/2)bh, Pythagoras a²+b²=c²
  • Sum of first n natural numbers = n(n+1)/2, Sum of squares = n(n+1)(2n+1)/6
🧠 Analytical Reasoning — Top 10 Topics
  • Number series — check differences first, then ratios, then squares/cubes/Fibonacci
  • Letter series — alternate letters, reverse alphabet (A=Z, B=Y, A+Z=27)
  • Coding-decoding — letter shift (+1/+2), number codes (A=1, B=2...Z=26)
  • Analogy — Doctor:Hospital, Fish:Water, Book:Library type pairs
  • Odd one out — 2D vs 3D shapes, prime vs composite, mammals vs birds
  • Blood relations — draw family tree, Mother's brother=Uncle, Sibling's child=Nephew/Niece
  • Ranking — Position from right = (Total+1) - Position from left
  • Direction sense — North/South/East/West turns, final direction after multiple turns
  • Syllogisms — All A are B + Some B are C → Some A may be C
  • Clock problems — Angle = |30H - 5.5M| degrees
📖 English Language — Top 10 Topics
  • Synonyms: Abundant=Plentiful, Benevolent=Kind, Candid=Frank, Diligent=Hardworking, Eloquent=Fluent
  • Antonyms: Ancient↔Modern, Courage↔Cowardice, Genuine↔Fake, Humble↔Arrogant, Transparent↔Opaque
  • Subject-verb agreement — Neither/Either/Each/Everyone takes SINGULAR verb
  • Tenses — "yesterday" needs simple past, "since/for" needs present perfect
  • Articles — "an" before vowel SOUND (an honest man, an hour, an MBA)
  • Prepositions — good AT, arrive ON Monday, interested IN, afraid OF
  • Idioms: Break the ice=start conversation, Bite the bullet=endure bravely, Burn midnight oil=work late
  • One-word substitution: Omniscient=knows everything, Incorrigible=cannot be corrected
  • Error spotting — double comparative (more smarter→smarter), wrong tense with time words
  • Passive voice — Active: She writes a letter → Passive: A letter is written by her

📐 QA Shortcuts
TRIG TABLE — Memorize Once
0° 30° 45° 60° 90° sin 0 1/2 1/√2 √3/2 1 cos 1 √3/2 1/√2 1/2 0 tan 0 1/√3 1 √3 ∞ TRICK: sin row = √0/2, √1/2, √2/2, √3/2, √4/2 cos row = sin in REVERSE order
LOG SHORTCUTS
log(1)=0 log(10)=1 log(2)≈0.301 log(3)≈0.477 log(5)≈0.699 log(aⁿ)=n·log(a) log(ab)=log(a)+log(b) log(a/b)=log(a)-log(b)
PERMUTATION vs COMBINATION
See "ARRANGE/ORDER/RANK" → Permutation nPr = n!/(n-r)! See "SELECT/CHOOSE/PICK" → Combination nCr = n!/(r!×(n-r)!) 5P2 = 5×4 = 20 5C2 = 5×4/2×1 = 10
PERCENTAGE SHORTCUTS
10% of X = X/10 5% = half of 10% 25% = X/4 33.33% = X/3
🧠 Reasoning Shortcuts
NUMBER SERIES METHOD
Step 1: Find differences (D1, D2, D3...) Step 2: Equal differences → AP series Step 3: Differences increasing by same amount → quadratic Step 4: Check squares: 1,4,9,16,25,36... Step 5: Check cubes: 1,8,27,64,125... Step 6: Check Fibonacci: each = sum of previous two Step 7: Check doubling/halving
CLOCK ANGLE FORMULA
Angle = |30H - 5.5M| At 3:00 → |90 - 0| = 90° At 6:30 → |180 - 165| = 15°
RANKING TRICK
Position from right = (Total + 1) - Position from left 8th from left in 20 students = 20+1-8 = 13th from right
CODING-DECODING
Type 1 — Letter Shift: A→C means +2, apply same shift to all Type 2 — Reverse: A=Z, B=Y (use formula: 27 - position) Type 3 — Number: A=1, B=2 ... Z=26
